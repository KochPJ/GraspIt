## Overview
[Omniverse Isaac Sim](https://developer.nvidia.com/isaac/sim) simluation of cluttered tabletops with 6DoF-pose estimation and testing. Both the scene sampler and the graps simulation can be run indepentendly 
Both the Scene Sampler and the Grasp Simulation can be run independently of each other. All information and requirements for the respective data sets can be found in the respective subsections.
